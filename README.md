<h1>Описание </h1>
Данный проект представляет собой веб-приложение, разработанное на React.

Главная цель приложения - генерация пароля в соответствии с параметрами, указанными пользователем.

<h2>Используемые технологии</h2>
<ul>
   <li>HTML</li>
   <li>CSS</li>
   <li>SCSS</li>
   <li>TypeScript</li>
   <li>React</li>
   <li>Formik</li>
</ul>
